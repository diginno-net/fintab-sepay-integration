# BE-001 Create Pancake Order Filter Builder

Create `backend/src/modules/pancake/pancake-order-filter.ts`.

Required exports:

- `PancakeOrderFilterInput`
- `PancakeOrderListQuery`
- `buildOrderListQuery(input)`
- `buildCompletedOrderQuery(input?)`

Rules:

- Prefer `filter_status[]` over scalar `status`.
- Prefer Unix `startDateTime`/`endDateTime` when `updateStatus` is present.
- Preserve legacy `date_from`/`date_to` fallback.
- Emit no undefined values.
- Do not include API keys.
