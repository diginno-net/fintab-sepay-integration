# Task: BE-001 - Fix worker job type compatibility

## Metadata
- Status: completed
- Estimate: 25m
- Depends on: -

## Files to Modify
- `backend/src/queue-worker.ts` (MODIFY)

## Description
Fix worker để xử lý đúng job type mà invoice service đang enqueue hiện tại, đồng thời giữ tương thích với các alias cũ đã có trong worker.

## Requirements
- Worker phải xử lý create draft khi background job type là `create_draft` hoặc alias `invoice:create-draft`.
- Worker phải xử lý poll create khi background job type là `poll_create` hoặc alias `invoice:poll-create` nếu có job loại này.
- Worker phải xử lý issue khi background job type là `issue` hoặc alias `invoice:issue`.
- Worker phải xử lý poll issue khi background job type là `poll_issue` hoặc alias `invoice:poll-issue` nếu có job loại này.
- Giữ logging hiện tại đủ để debug job id, type, invoice job id.
- Không thay đổi schema DB.

## Context
Hiện `invoice-job.service.ts` enqueue job type `create_draft` và `issue`, nhưng `queue-worker.ts` chỉ nhận `invoice:create-draft` và `invoice:issue`. Đây là blocker làm draft job không tạo hóa đơn thật trên SePay khi worker chạy.

## Verification
```bash
cd backend
npm run typecheck
```

Manual:
```bash

npm run dev:worker
```

Tạo draft từ UI và xác nhận worker không log `unknown job type` cho job `create_draft`.
