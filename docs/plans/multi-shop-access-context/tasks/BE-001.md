# BE-001 Shop Access Service

Add `backend/src/modules/tenant/shop-access.service.ts`.

Exports:

- `listAccessibleShops`
- `assertUserCanAccessShop`
- `resolveCurrentShopId`
- `setSessionCurrentShop`
- `accessibleShopIds`

Behavior:

- User access is driven by `user_shop_access`.
- Inaccessible shops return `FORBIDDEN`.
