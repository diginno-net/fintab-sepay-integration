## Description
Add backend APIs for admins/owners to assign users to shops.

## Requirements
- Add permissions: `shop_access:read`, `shop_access:write`.
- Add endpoint to list users with their shop assignments.
- Add endpoint to get one user's shop assignments.
- Add endpoint to replace one user's shop assignments and default shop.
- Validate default shop belongs to assigned shops.
- Prevent normal users from assigning shop access.
