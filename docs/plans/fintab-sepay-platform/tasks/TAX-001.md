## Description
Implement product tax profiles and shop tax defaults used to map Pancake/Fintab product lines into SePay invoice tax fields.

## Requirements
- Create `product_tax_profiles` persistence and service.
- Create `shop_tax_defaults` persistence and service.
- Support tax rates: `-2`, `-1`, `0`, `5`, `8`, `10`.
- Support tax categories: `taxable`, `non_taxable`, `non_declarable`, `zero_rated`.
- Support invoice line types: `1`, `2`, `3`, `4`.
- Support unknown product policies: `warn`, `block`, `use_default`.
- Product tax profiles can be tenant-level defaults or shop-level overrides.
- All reads/writes must be tenant scoped and shop guarded.

## API Endpoints

```text
GET  /v1/products/:id/tax-profile
PUT  /v1/products/:id/tax-profile
GET  /v1/shops/:shopId/tax/defaults
PUT  /v1/shops/:shopId/tax/defaults
POST /v1/products/tax-profiles/import
```
