## Description
Build UI for shop management with per-shop settings. Each shop has its own Pancake and SePay configuration tabs.

## Requirements

### Shop List (/shops)
- List all shops for current tenant
- Show connection status (Pancake connected, SePay configured)
- Actions: create shop, edit, delete
- Empty state if no shops

### Shop Settings Hub (/shops/:shopId/settings)
- Tab navigation: Overview | Pancake | SePay | Tax | Webhook | Rules
- Each tab is a separate page/route

### Pancake Config Tab (/shops/:shopId/settings/pancake)
- shop_id (text input)
- shop_name (text input)
- api_key (write-only input, never shown after save)
- webhook_secret (write-only input, never shown after save)
- allowed draft statuses (multi-select)
- allowed issue statuses (multi-select)
- Test connection button: loading/success/error states
- Save only if test passes or user explicitly bypasses

### SePay Config Tab (/shops/:shopId/settings/sepay)
- env selector: sandbox | production
- client_id (text input)
- client_secret (write-only input, never shown after save)
- After saving client credentials, auto-fetch provider accounts
- provider_account_id (selector from fetched accounts)
- template_code (selector: GTGT=1 | Bán hàng=2)
- invoice_series (text input or selector from accounts)
- default payment_method (selector)
- default tax_rate (number input for GTGT)
- usage/quota display (read-only)
- Test connection button
- BLOCK issue button if config incomplete: show warning banner with CTA

### Tax Config Tab (/shops/:shopId/settings/tax)
- default_tax_rate selector: -2 | -1 | 0 | 5 | 8 | 10
- default_invoice_unit input
- default_invoice_type selector: GTGT | Bán hàng
- unknown_product_policy selector: warn | block | use_default
- summary counters: mapped products, missing tax profile, using default tax
- CTA to product tax mapping screen

### Secrets Policy
- Pancake api_key and webhook_secret: write-only, show masked "••••••••" after save
- SePay client_secret: write-only, show masked "••••••••" after save
- Config GET response: only has_client_secret: boolean, has_api_key: boolean, last_updated_at

### Test Connection States
- Loading: button shows "Đang kiểm tra..."
- Success: green checkmark + "Kết nối thành công"
- Error: red X + error message from API
